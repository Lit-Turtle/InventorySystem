# InventorySystem
Python practice project.

Allows user to track inventory with category and products. Track the profit and revenue, while also having a local json save file to login to inventory.
Can also generate a sales report on top products and categories.

users.json - user data.
Sales Report.txt - where the sales report is written.
main.py - contains the inventory, category, and product classes.
user_comp - Contains user menus and other methods.
run_inventory.bat - start of program, call on it to run program as intended.
